# DBD-perk-overay
데바데 퍽 설명을 화면에 표시하는 프로그램입니다.

데바데 우측 하단의 퍽 이미지를 인식하여 설명을 불러옵니다.

# 사용법
데바데를 실행 후 프로그램을 실행합니다.

# 설정
settings.txt 파일을 수정 후 재시작합니다.

# 제약사항
주 모니터에서, FHD QHD 해상도에서 작동합니다.
